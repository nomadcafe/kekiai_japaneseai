# BGM素材库

このディレクトリには動画生成で使用する背景音楽（BGM）ファイルを配置します。

## 使用方法

1. BGMファイル（MP3、WAV、M4Aなど）をこのディレクトリに配置
2. 動画生成時に、UIでBGMを有効化し、ファイル名を指定
3. 例: `bgm/background_music.mp3` を配置した場合、UIで `background_music.mp3` と入力

## 推奨設定

- **ファイル形式**: MP3、WAV、M4A
- **音量**: 動画生成時に調整可能（デフォルト: 15%）
- **長さ**: 短いループ可能なファイルが推奨（自動的に動画の長さに合わせてループされます）

## 注意事項

- BGMファイルは著作権フリーのものを使用してください
- 商用利用する場合は、使用許諾を確認してください
